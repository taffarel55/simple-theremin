# Teremim

Um teremin simples usando uma placa de desenvolvimento `Arduino`, um sensor de distância `HC-SR04` e um `Buzzer`.

## Montagem

Tenha atenção aos pinos, como mostra o código, ECHO = 6, TRIGGER = 7 e BUZZER = 2
```c++
const int echoPin = 6; 
const int trigPin = 7;
const int buzzer = 2;
```
